How to run
===========
Install dependencies: `yarn`

Run: `npx parcel serve index.html` 


Libraries used
===============
- **Bundler**: Parcel
- **Frontend Framework**: React
- **Routing**: Reach Router
- **Design Framework**: Tailwind CSS
- **Charts**: Chart.js and react-chartjs-2
- **Other**:
    - Ramda
    - date-fns
